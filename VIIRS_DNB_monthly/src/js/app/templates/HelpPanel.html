Navigating the Map
  • Click and drag or use arrow keys to pan
  • Mouse scroll forward or use + key to zoom in
  • Mouse scroll backward or use - key to zoom out
Time-enabled Imagery
  • Select the desired month using the time slider.
  • Automatically play back the sequence using the play button.
  • The data begin in Apr. 2012, but images including stray-light impacted data are only available from Jan. 2014 to the present.
Querying for Data Values
  • Click anywhere on the map to retrieve the pixel value and the latitude/longitude at that point.
  • Depending on which layer is visible, the value shown either represents average radiance (nanowatts/cmÂ²/sr) or number of cloud-free observations for that pixel.
