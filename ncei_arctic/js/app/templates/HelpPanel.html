Navigating the Map
  • Click and drag or use arrow keys to pan
  • Mouse scroll forward or use + key to zoom in
  • Mouse scroll backward or use - key to zoom out
Identifying Features and Querying Rasters
You have several options to identify features and/or query raster values within the layers currently visible:
  • Single-click on the map
  • Or, choose another tool from the "Identify" menu:
  • Click on to draw a rectangle
  • Click on to draw a polygon
  • Click on to enter coordinates for a bounding box
Note: identifying on the WMS raster layers can only be done with a single-point click.
A popup will appear with a list of the selected features or raster values. Mouse-over the list to highlight features on the map (only works with some layers). Click the magnifying glass icon to zoom to that feature. Click on a row to view attributes for that feature.

Map projection: WGS84 / UPS North; EPSG:32661

These data not to be used for navigation. For official navigation products, please refer to the U.S. nautical charts available from the NOAA Office of Coast Survey. https://www.nauticalcharts.noaa.gov
