Description:
Abstract:
A line file representing locations of the pipeline infrastructure in the Gulf of Mexico associated with the oil and gas industry is presented. These layers were modified from GIS data acquired from the U. S. Department of the Interior Bureau of Ocean Energy Management (BOEM) website. We note that pipelines are being added or modified continuously, thus regular updates of this database are required to know the true distribution of pipelines. Although file description reports are available, no FGDC-compliant metadata are available for the original files from BOEM. The current layers are non-projected with coordinates in decimal degrees. (www.gomr.mms.gov)
Purpose:
This data layer was developed to replace an older dataset that was outdated. The oil and gas pipeline data is used for reference in maps showing coastal features in the Gulf of Mexico.

Completeness_Report:
The information contained within this dataset is the same as the compiled dataset created by the Bureau of Ocean Energy Management except for the omission of the following fields: FNODE_, TNODE_, LPOLY_, RPOLY_, LENGTH, PIPELINES#, and PIPELINES-ID. Process_Step:
Process_Description:
The ArcInfo interchange file (E00) was downloaded from the BOEM website. ArcCatalog was used to import the interchange file and convert it to a coverage. The features were then loaded into ArcSDE. Some attribute fields that existed in the coverage were omitted in the ArcSDE layer because they were artifacts from ArcInfo. These fields included FNODE_, TNODE_, LPOLY_, RPOLY_, LENGTH, PIPELINES#, and PIPELINES-ID.
