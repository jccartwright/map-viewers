Description:
Abstract:
Polygon data for all major shipping lanes associated with all ports in the Gulf of Mexico are presented. These layers were modified from GIS data acquired from the U. S. Department of the Interior Minerals Management Service (MMS) website. Although file description reports are available, no FGDC-compliant metadata are available for the original files from MMS. The current layers are non-projected with coordinates in decimal degrees. (www.mms.gov)
Purpose:
This data layer was developed to replace an older dataset that was outdated. The shipping fairways data is used for reference in maps showing coastal features in the Gulf of Mexico.

Process_Step:
Process_Description:
The ArcInfo interchange file (E00) was downloaded from the MMS website. ArcCatalog was used to import the interchange file and convert it to a coverage. The features were then loaded into ArcSDE. Some attribute fields that existed in the coverage were omitted in the ArcSDE layer because they were artifacts from ArcInfo. These fields included AREA, PERIMETER, and FAIRWAY_.
