Description:
Abstract:
This data set is a subset of the Subcuencas hidrologicas layer for the Mexican states bordering the Gulf of Mexico. This data set was derived from the nation-wide hydrologic unit boundary layer produced by the Comision Nacional para el Conocimiento y Uso de la Biodiversidad (CONABIO).
Purpose:
This hydrologic unit boundaries layer was developed for use in a Gulf of Mexico Atlas to show watersheds in the Gulf region.
