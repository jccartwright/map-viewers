Description:
Abstract:
The 1992 National Land Cover Dataset (NLCD) coverage for watersheds in Mobile and Baldwin Counties, Alabama was used as an input in a hydrologic modeling system to determine the effects of land cover land used change on shallow water aquatic ecosystems. The dataset includes eleven land cover land use classes.
Purpose:
The 1992 NLCD data was an input for watershed modeling historical simulations for watersheds contiguous to the Mobile Bay estuary.
Supplemental_Information:
The 1992 NLCD was used as an input for hydrologic models.

Process_Step:
Process_Description:
The 1992 NLCD was downloaded from the EROS data center using the Earth Explorer web retrieval tool and reprojected to UTM zone 16, NAD83. ArcGIS was used to subset the coverage to Mobile and Baldwin Counties. The land cover land use classes were remapped to a common classification with the 2001 NLCD to facilitate the comparison of change between these two time periods. The merging of classes to reach the final classification resulted in minor changes from the original classes used in development of the 1992 NLCD.

Entity_and_Attribute_Information:
Overview_Description:
Entity_and_Attribute_Overview:
This NLCD 1992 land cover land use raster dataset has been mapped to a common classification with the 2001 NLCD data for use in hydrologic modeling. The 1992 land cover land use data contains eleven (11) classes: water, low intensity/Recreational, medium/high density residential, urban commercial, bare soil/transitional, deciduous forest, evergreen forest, mixed forest/shrub, agricultural/pasture, woody wetlands, and emergent herbaceous wetlands.
