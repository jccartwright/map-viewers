Description:
Abstract:
Point location for oil and gas installations in federal waters of the Gulf of Mexico are presented. This layer was converted from GIS data acquired from the U. S. Department of the Interior Bureau of Ocean Energy Management (BOEM) website. We note that platforms are being added and removed continuously, thus regular updates of this data base are required to know the true distribution of platforms. Although file description reports are available, no FGDC-compliant metadata are available for the original files from BOEM. The current layers are non-projected with coordinates in decimal degrees.(http://www.boem.gov)
Purpose:
This data layer was developed to replace an older dataset that was outdated. The oil and gas installation data is used for reference in maps showing coastal features in the Gulf of Mexico.

Completeness_Report:
The information contained within this dataset is the same as the compiled dataset created by the Bureau of Ocean Energy Management except for the omission of the following fields: AREA, PERIMETER, PLATFORMS#, PLATFORMS-ID, POLYGONID, SCALE, and ANGLE. These fields were dropped because BOEM indicates that they "are ArcInfo internal control parameters and have no applicable relevance to the use of the data".

Process_Step:
Process_Description:
The ArcInfo interchange file (E00) was downloaded from the BOEM website. ArcCatalog was used to import the interchange file and convert it to a coverage. The features were then loaded into ArcSDE. Some attribute fields that existed in the coverage were omitted in the ArcSDE layer because they were artifacts from ArcInfo. These fields included AREA, PERIMETER, SCALE, and ANGLE.
