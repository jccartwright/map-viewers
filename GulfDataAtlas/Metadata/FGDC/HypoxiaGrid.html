Description:
Abstract:
This hypoxia dataset is a 10 minute grid based on 2001-2011 Hypoxia Watch stations. The grid has attributes added that allow analysis of hypoxia in the Gulf of Mexico. Such analysis might include frequency of data collection, frequency of hypoxia, years data collection occurred, etc. Stations that recorded oxygen values less than or equal to 2.0 mg/L are considered hypoxic.
Purpose:
This dataset was developed to include in the Gulf of Mexico Data Atlas to show examples of Hypoxia Studies.
Supplemental_Information:
The oxygen values in the grid attribute table have been binned into 0.2 mg/L ranges.

Source_Contribution:
Hypoxia Watch stations provided the locations and oxygen values that were used to populate the grid attributes.
Process_Step:
Process_Description:
A fishnet grid was created covering an area between -86 to -98 degrees longitude and 25 to 31 degrees latitude. The grid was spaced in 10 minute intervals. The grid was then spatially joined with a combined Hypoxia Watch station layer to get a count of stations falling within each grid cell. Grid cells with no stations occurring in them were deleted. The Intersect tool was used on the station layer to add the grid cell ID to each station. This provided a cross reference to compare the stations with the grid. Stations that fell on a grid line or corner were originally counted in two or more grid cells. To avoid duplication, the count was adjusted to include the station only in the closest grid to the north and/or east. Attributes were added and populated based on the oxygen values in the station layer.
