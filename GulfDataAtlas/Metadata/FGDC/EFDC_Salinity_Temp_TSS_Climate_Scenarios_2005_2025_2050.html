Description:
Abstract:
This dataset includes Environmental Fluid Dynamics Code (EFDC) hydrodynamic model outputs for Mobile Bay, AL. The model was run to generate total suspended sediment (TSS), salinity, and temperature values for the month of May in 2005, 2025, and 2050 climate scenarios. The EDFC model used freshwater inflow, temperature and TSS data at 22 discharge points around Mobile Bay, which were oupputs of the Loading Simulation Program in C++ (LSPC) as inputs. Meterological, bathymetric and water surface elevation data were also used as inputs to EDFC model. The EDFC model gives four depth layer output values for each cell in a horizontal grid covering the extent of Mobile Bay.
Purpose:
These data were generated as part of a study to explore the impact of climate change on salinity, temperature, TSS and light attenuation in Mobile Bay and its effect on submerged aquatic vegetation (SAV) and seagrasses in Mobile Bay.

Process_Step:
Process_Description:
The EFDC hydrodynamic model was run for 3 climate scenarios (2005, 2025, 2050) using the following input data: 1. Outputs of the LSPC watershed model (freshwater inflow, TSS, and temperature at 22 discharge points around Mobile Bay for each climate scenario), 2. Meteorological data (NOAA NCDC), 3. Bathymetric data (NOAA NGDC), and 4. Water surface elevation data (NOAA CO-OPS). The EDFC model generated TSS, salinity and temperature values for each of the climate scenarios output in a horizontal grid with 4 depth layers for each cell in the grid. The Environmental Fluid Dynamics Code (EFDC Hydro) is a state-of-the-art hydrodynamic model that can be used to simulate aquatic systems in one, two, and three dimensions. For more information on the EFDC model, please see http://www.epa.gov/athens/wwqtsc/html/efdc.html.

Entity_and_Attribute_Information:
Overview_Description:
Entity_and_Attribute_Overview:
The EDFC model output data are provided for each horizontal grid cell within the modeling domain of Mobile Bay and for 4 vertical layers. There are 99 fields in the grid containing the following statistics: 1. Mean (over the month of May, 2005) for each horizontal cell in each layer. 2. Standard deviation (over the month of May, 2005) for each horizontal cell in each layer. 3. Max (over the month of May, 2005) for each horizontal cell in each layer. 4. Mean of the entire water column (all layers). 5. Standard deviation of the entire water column (all layers). The last two fields contain the fields' definitions.
