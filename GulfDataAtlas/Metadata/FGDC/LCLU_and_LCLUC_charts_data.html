Description:
Abstract:
The areal extent of Land Cover Land Use (LCLU) classes for Mobile and Baldwin Counties, AL were extracted from LCLU maps and plotted as bar graphs to show changes in LCLU classes between four time steps: 1948, 1992, 2001, and 2030. The LCLU classification for 2030 was generated using the Prescott Spatial Growth Model (PSGM).
Purpose:
Evaluate the Land Cover Land Use (LCLU) class changes over time in order to provide information for hydrodynamic modeling for the Mobile Bay watershed.

Process_Step:
Process_Description:
LCLU scenarios were developed for 1992 and 2001 from the NLCD and in 2030 using the Prescott Spatial Growth Model (PSGM). A map of 1948 land cover was used to represent a “green” pre-development scenario. The areal extents of the LCLU classes and the changes in class area for each scenario are shown here in tabular and bar graph formats.
