Description:
Abstract:
This dataset is a compilation of 2001-2011 Hypoxia Watch summer sampling stations. Stations that recorded oxygen values less than or equal to 2.0 mg/L are considered hypoxic.
Purpose:
This dataset was developed to include in the Gulf of Mexico Data Atlas to show examples of Hypoxia Studies.
Supplemental_Information:
The oxygen values in the attribute table have been binned into 0.2 mg/L ranges.
