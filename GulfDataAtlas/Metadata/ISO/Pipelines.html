      Abstract: 
          A line file representing locations of the pipeline infrastructure in the Gulf of Mexico associated with the 
          oil and gas industry is presented. These layers were modified from GIS data acquired from the U. S. 
          Department of the Interior Minerals Management Service (MMS) website. We note that pipelines are being added 
          or modified continuously, thus regular updates of this database are required to know the true distribution 
          of pipelines. Although file description reports are available, no FGDC-compliant metadata are available for 
          the original files from MMS. The current layers are non-projected with coordinates in decimal degrees. 
          (www.gomr.mms.gov)
      Purpose: 
          This data layer was developed to replace an older dataset that was outdated. The oil and gas pipeline data is 
          used for reference in maps showing coastal features in the Gulf of Mexico.
