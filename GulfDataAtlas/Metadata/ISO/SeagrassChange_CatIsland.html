          This classification raster file with color tables and class descriptions was produced for NOAA and the 
          University of Southern Mississippi to provide an accurate visual historical record of changes in seagrass 
          beds adjacent to the Mississippi barrier islands. It is designed for remote sensing, GIS, and historical 
          analysis for seagrass trends for the island and the time period (years) indicated in the file name.

      Report: 
        Conceptual consistency: 
          Measure description: 
              Each classification raster file in the series was created by comparing overlapping pixel values indicating 
              the presence or absence of seagrass and the trends of "Seagrass Gain", "Seagrass Loss" or "Seagrass 
              Unchanged" for each pixel was determined. The classification raster files have color tables and Class 
              name associations. Seagrass Loss = red, Seagrass Gain = blue and Seagrass Unchanged = green. This 
              process was consistent regardless of image type or acquisition date.
