      Abstract: 
          This dataset includes Environmental Fluid Dynamics Code (EFDC) hydrodynamic model outputs of salinity and 
          temperature for Mobile Bay, AL using 1992 and 2001 Land Cover Land Use (LCLU) scenarios. This dataset 
          includes model outputs for the period of May 2005 from separate model runs. The EDFC model used freshwater 
          inflow, temperature and TSS data at 22 discharge points around Mobile Bay (outputs of the Loading Simulation 
          Program in C++ [LSPC++]) as inputs. Meterological, bathymetric and water surface elevation data were also 
          used as inputs to EDFC model. The EDFC model produced a horizontal grid covering the extent of Mobile Bay 
          with 99 fields of hydrodynamic statistics for salinity and temperature for the 2001 and 1992 LCLU scenarios 
          and their differences (2001-1992). Each cell in the horizontal modeling grid contains values for four depth 
          layers. The major stats provided are: 1. Mean (over the month of May, 2005) for each horizontal cell in each 
          layer. 2. Standard deviation (over the month of May, 2005) for each horizontal cell in each layer. 3. Max 
          (over the month of May, 2005) for each horizontal cell in each layer. 4. Mean of the entire water column 
          (all layers). 5. Standard deviation of the entire water column (all layers). The last two fields contain the 
          fields' definitions
      Purpose: 
          These data were produced as part of a study to explore the impact of climate change on salinity, temperature, 
          TSS and light attenuation in Mobile Bay and its effect on submerged aquatic vegetation (SAV) and seagrasses 
          in Mobile Bay.

              Description: 
                  The EFDC hydrodynamic model was run for 1992 and 2001 land cover land use (LCLU) scenarios using the 
                  following input data: 1. Outputs of the LSPC++ watershed model (freshwater inflow, TSS, and 
                  temperature at 22 discharge points around Mobile Bay for each LCLU scenario), 2. Meteorological data 
                  (NOAA NCDC), 3. Bathymetric data (NOAA NGDC), and 4. Water surface elevation data (NOAA CO-OPS). The 
                  EDFC model generated TSS, salinity and temperature values for each of the climate scenarios output 
                  in a horizontal grid with 4 depth layers for each cell in the grid. The Environmental Fluid Dynamics 
                  Code (EFDC Hydro) is a state-of-the-art hydrodynamic model that can be used to simulate aquatic 
                  systems in one, two, and three dimensions. For more information on the EFDC model, please see 
                  http://www.epa.gov/athens/wwqtsc/html/efdc.html.
