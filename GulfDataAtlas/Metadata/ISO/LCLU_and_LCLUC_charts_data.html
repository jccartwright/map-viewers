      Abstract: 
          The areal extent of Land Cover Land Use (LCLU) classes for Mobile and Baldwin Counties, AL were extracted from 
          LCLU maps and plotted as bar graphs to show changes in LCLU classes between four time steps: 1948, 1992, 
          2001, and 2030. The LCLU classification for 2030 was generated using the Prescott Spatial Growth Model (PSGM).
      Purpose: 
          Evaluate the Land Cover Land Use (LCLU) class changes over time in order to provide information for 
          hydrodynamic modeling for the Mobile Bay watershed.
