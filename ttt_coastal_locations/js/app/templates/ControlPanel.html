Estimated Tsunami Travel Times to Coastal Locations

This map displays estimated pre-computed tsunami travel times to select coastal locations from any point in the ocean. Travel times (in hours), were calculated with TTT Software (P. Wessel, Geoware) using bathymetry derived from the 2 arc-minute ETOPO2 grid.
More information about tsunami travel time maps.

Important: this map is for general information only, and is not to be used in the event of an actual tsunami. There may be significant sources of error in the travel time calculations. Please visit tsunami.gov for real-time tsunami information.
