Help with map navigation using mouse:

  • Click and drag the map to pan
  • SHIFT + Click to recenter
  • SHIFT + Drag to zoom in to a box
  • SHIFT + CTRL + Drag to zoom out
  • Mouse Scroll Forward to zoom in
  • Mouse Scroll Backward to zoom out
  • Use Arrow keys to pan
  • + key to zoom in a level
  • - key to zoom out a level
  • SHIFT + Double Click to Center and Zoom in

Identifying features:

  • Single-click on the map to identify features (operates on visible layers).
  • A popup will appear with a list of the features near the click point.
  • Mouse-over the list to highlight features on the map.
  • Click the magnifying glass icon to zoom to the extent of a feature.
  • Click on a row to view attributes for that feature.
  • : Identify features by drawing a rectangle on the map.
  • : Identify features by entering coordinates to define a bounding box.

Searching for Data:

  • Search for Data: opens a dialog where you can specify a date range, cruise ID, instrument, number of beams, recording range, and swath width.\ The lines visible on the map will be filtered based on this input criteria.
  • Reset: clears the current filter.
